📌 Sobre o Projeto

Este repositório contém exercícios de Banco de Dados com foco em:

- Modelagem Conceitual

- Modelagem Lógica

- Criação de Dicionário de Dados

Os modelos foram desenvolvidos utilizando o software BRModelo (ferramenta para modelagem de banco de dados) e a documentação foi organizada em Word e PDF.

🎯 Objetivo

O objetivo deste projeto é praticar e consolidar os seguintes conceitos:

- Identificação de entidades e atributos

- Definição de relacionamentos

- Cardinalidade

- Normalização

- Transformação do modelo conceitual para o modelo lógico

- Estruturação de dicionário de dados

🛠️ Ferramentas Utilizadas

- BRModelo

- Microsoft Word

- PDF

🚀 Como Utilizar

1. Abra os arquivos do modelo conceitual no BRModelo.

2. Analise a conversão para o modelo lógico.

3. Consulte o dicionário de dados para entender a estrutura das tabelas.

4. Utilize os arquivos em Word ou PDF para estudo e revisão.
